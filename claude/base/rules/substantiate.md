# Substantiate every claim

For every factual claim about code, services, or data, provide verifiable evidence. If you can't produce one, label the claim "unverified."

## Evidence by claim type

- **Code** → source repo link with file path and line range (e.g. GitHub, GitLab)
- **API behavior** → curl command + actual response, or official API reference link
- **Developer docs** → link to official documentation (e.g. AWS docs, MDN, language reference)
- **Metrics / stats** → query + result, or a link to a live dashboard or chart
- **Third-party behavior** → link to changelog, issue tracker, or release notes
